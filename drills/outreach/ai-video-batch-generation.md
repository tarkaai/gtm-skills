# AI Video Batch Generation

This drill generates personalized prospect videos at scale using AI avatar and voice-clone platforms (HeyGen, Tavus, or Sendspark) instead of manually recording each video. The sender records a single template video or trains an AI avatar once, then the system generates unique personalized videos for each prospect by inserting their name, company, pain point, and showing their website as the video background.

This is the core differentiator of the `video-prospecting-scale` play versus `outbound-with-short-video`, which relies on founder-recorded Loom videos for each prospect.

## Input

- Enriched prospect list in Clay with: first_name, company, website_url, role, signal/pain_point
- Trained AI avatar or recorded template video (one-time setup)
- AI video platform account (HeyGen, Tavus, or Sendspark)
- Instantly account with warmed sending domains
- Cal.com booking link for video CTAs

## Steps

### 1. Prepare the prospect batch in Clay

Open your Clay table with the target prospects. Ensure these columns are populated using the `clay-enrichment-waterfall` fundamental:

- `first_name`, `last_name`, `email`
- `company`, `website_url`
- `role` / `title`
- `signal` (recent event: funding, hiring, tech adoption, job change)
- `pain_point` (mapped from signal via Clay AI formula)

Add a Clay AI formula column that generates the personalized video script for each prospect:

```
Template: "Hey {first_name}, this is {sender_name} from {sender_company}. I noticed {company} just {signal}. {pain_point_connection}. We helped {similar_company} achieve {result} in {timeframe}. I'd love to show you how -- there's a link below to grab 15 minutes."
```

Cap each script at 80 words / 60 seconds of spoken content.

### 2. Choose your AI video platform

Select based on volume and personalization depth:

- **Sendspark** ($99/mo): Best for high volume. Record once, name-swap + dynamic website background. Fastest generation. Use `sendspark-personalized-video` fundamental.
- **Tavus** ($199/mo): Best for natural feel. Full lip-sync with cloned voice speaks each unique script. Use `tavus-personalized-video` fundamental.
- **HeyGen** ($330/mo): Best for multilingual or fully scripted. Text-to-avatar with custom background per video. Use `heygen-personalized-video` fundamental.

For Smoke/Baseline: Sendspark is sufficient and most cost-effective.
For Scalable/Durable: Tavus or HeyGen for deeper per-prospect personalization.

### 3. Train the avatar / record the template (one-time)

**For Sendspark:** Record a 60-90 second template video following the outreach script structure. Say "[First Name]" where you want name personalization. Upload via API.

**For Tavus:** Record a 2-minute training video of natural speech. Upload to create a replica. Then the API generates unique videos from any script.

**For HeyGen:** Upload a 2-5 minute training video. The API creates a digital avatar that can speak any text.

**Human action required:** The founder/sender must record the initial training video or template. This is a one-time recording -- all subsequent videos are generated by AI.

### 4. Generate the video batch via n8n

Build an n8n workflow using `n8n-workflow-basics` that processes the Clay prospect table:

1. **Trigger:** Manual trigger or scheduled cron (e.g., Monday 6am for the week's batch)
2. **Step 1:** Fetch the prospect batch from Clay via API (filter: `video_generated = false`)
3. **Step 2:** For each prospect, call the video platform API to generate a personalized video:
   - Pass the script from the Clay AI formula column
   - Pass the prospect's website URL as background
   - Pass the Cal.com booking link as CTA
4. **Step 3:** Store the returned `video_id` back in Clay
5. **Step 4:** Poll for completion (or receive webhook callbacks)
6. **Step 5:** When each video is ready, update Clay with: `video_url`, `thumbnail_url`, `gif_url`, `video_generated = true`

Rate limits: Process 5-10 concurrent video generations. Total batch of 100 videos takes 2-5 hours depending on platform.

### 5. Build the email campaign in Instantly

Once videos are generated, export the Clay table to CSV with columns mapped to Instantly merge fields:

| email | first_name | company | video_share_url | video_gif_url |
|-------|-----------|---------|-----------------|---------------|

Create a 3-step email sequence in Instantly using `instantly-campaign` and `loom-embed-in-email` (the embed technique works identically for any hosted video GIF):

**Email 1 (Day 0) -- Video email:**
```
Subject: {first_name}, recorded something for you
Body:
Hey {first_name},

I put together a quick 60-second video for {company}:

[Embedded GIF thumbnail linked to video share URL]

{one_line_context_from_clay}

If it clicks, there's a link in the video to grab 15 minutes.

{sender_name}
```

**Email 2 (Day 3) -- Text follow-up:**
```
Subject: Re: {first_name}, recorded something for you
Body:
Hey {first_name}, did you catch the video I sent?

Quick version: {value_prop_one_liner}.

Happy to walk through it live -- {cal_link}

{sender_name}
```

**Email 3 (Day 7) -- Breakup:**
```
Subject: Re: {first_name}, recorded something for you
Body:
{first_name} -- last note from me on this.

If {pain_point} isn't top of mind right now, totally understand. But if it is, the video is still here: {video_share_url}

{sender_name}
```

### 6. Log in CRM and track

Using `attio-contacts`, update each prospect in Attio:
- `video_type: ai_generated`
- `video_platform: {heygen|tavus|sendspark}`
- `video_url: {share_url}`
- `video_campaign: {campaign_slug}`
- `video_sent_date: {date}`

## Output

- Batch of AI-personalized videos (one per prospect) hosted on the video platform
- Cold email campaign in Instantly with embedded video thumbnails
- CRM records updated with video metadata
- Prospect-to-video mapping for engagement tracking

## Triggers

Run weekly: generate the next batch of videos on Monday, launch email campaigns on Tuesday. Typical cadence at scale: 100 videos/week.
