# Set Up OBS for Recording

### Step-by-step
1. Download and install OBS Studio from obsproject.com (free, open-source).
2. Run the Auto-Configuration Wizard: choose 'Optimize for recording' and select your preferred resolution (1080p recommended).
3. Set up your first Scene: a scene is a layout of sources (camera, screen, overlays).
4. Add sources: 'Display Capture' or 'Window Capture' for your screen, 'Video Capture Device' for your webcam.
5. Arrange sources: resize and position your webcam overlay, add a background image or branded frame.
6. Configure audio: add your microphone as an audio input source. Set noise suppression and gain filters.
7. Set recording format: go to Settings > Output > Recording. Use MKV format (crash-safe) and choose the encoder (x264 for CPU, NVENC for NVIDIA GPU).
8. Set recording quality: 'High Quality, Medium File Size' is a good default for most content.
9. Configure hotkeys: set Start/Stop Recording hotkeys so you don't need to click buttons during a presentation.
10. Do a test recording: record 1 minute, play it back, and verify video quality, audio levels, and webcam positioning.
