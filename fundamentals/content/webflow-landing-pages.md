# Build Webflow Landing Pages

### Step-by-step
1. In Webflow, create a new page or clone an existing landing page template.
2. Structure the page with conversion-focused sections: Hero (headline + CTA), Problem, Solution, Social Proof, Features, FAQ, and Final CTA.
3. Write the hero headline: focus on the main benefit. Support with a subheadline that explains the 'how'.
4. Add a clear CTA above the fold: 'Start Free Trial', 'Book a Demo', 'Download Guide'. Make the button visually prominent.
5. Add social proof: logos of customers, testimonial quotes, metrics ('500+ teams trust us'), or review badges.
6. Design for mobile first: over 50% of traffic is mobile. Ensure text is readable, buttons are tappable, and the form is easy to fill.
7. Add a lead capture form: keep it short (name + email for gated content, add company and role for demo requests).
8. Set up form submissions: connect the form to your email tool (Loops), CRM (Attio), or use webhooks to n8n.
9. Add tracking: install PostHog, Google Ads pixel, LinkedIn Insight Tag, and Meta Pixel on the page.
10. Publish and test: check the page loads fast (under 3 seconds), all links work, forms submit correctly, and tracking pixels fire.
