# Run Google Ads Search Campaigns

### Step-by-step
1. Create a new Search campaign in Google Ads with your objective (Leads or Sales).
2. Structure the campaign: create separate ad groups for each keyword theme (product features, competitor comparisons, pain points).
3. Write Responsive Search Ads: provide 8-15 unique headlines and 3-4 descriptions. Include keywords in headlines, benefits in descriptions.
4. Pin critical headlines: if a specific headline must always show (e.g., your brand name or main CTA), pin it to position 1.
5. Set up ad extensions: add sitelinks (4-6 links to key pages), callouts (trust signals like '500+ customers'), and structured snippets.
6. Add call extensions if you want phone leads, and location extensions for local businesses.
7. Set bid adjustments by device: if mobile converts differently than desktop, adjust bids accordingly.
8. Launch with a moderate budget and monitor the first week's performance closely.
9. Optimize weekly: pause low-performing keywords (high spend, no conversions), increase bids on winners, add new negative keywords.
10. Track quality scores: optimize ad relevance and landing page experience to lower CPCs. Quality Score 7+ is the target.
